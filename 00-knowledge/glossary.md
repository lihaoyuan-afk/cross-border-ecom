# 跨境电商通用术语表

> 最后更新：2026-06-13 | 词条数：50+

---

## 平台术语

| 术语 | 英文 | 解释 |
|------|------|------|
| FBA | Fulfillment by Amazon | 亚马逊配送服务，卖家将货物存入亚马逊仓库，由亚马逊负责发货和客服 |
| FBM | Fulfillment by Merchant | 卖家自发货，自行管理库存和物流 |
| BSR | Best Seller Rank | 畅销排名，数字越小代表销量越好，按品类独立计算 |
| ASIN | Amazon Standard Identification Number | 亚马逊商品唯一标识码，10位字母数字组合 |
| SKU | Stock Keeping Unit | 库存单位，卖家自定义的商品编码，用于区分不同规格 |
| Listing | — | 亚马逊商品详情页，包含标题、图片、价格、描述等信息 |
| Buy Box | — | 黄金购物车，抢到 Buy Box 的卖家会获得默认购买按钮，极大影响销量 |
| A+ Content | — | 品牌注册后可用的增强版商品描述，支持图文混排 |
| Storefront | — | 亚马逊品牌店铺页，需要注册品牌 |
| Vine | — | 亚马逊 Vine 计划，通过赠品换取真实 Review |
| Parent / Child ASIN | — | 父子 ASIN，父 ASIN 为变体商品的总页面，子 ASIN 为各具体规格 |
| MOQ | Minimum Order Quantity | 最小起订量，供应商要求的最低订货数量 |
| OEM | Original Equipment Manufacturer | 贴牌生产，按买方要求生产并使用买方品牌 |
| ODM | Original Design Manufacturer | 原始设计制造，供应商提供设计方案 |
| 独立站 | Independent Store | 品牌自建网站（如 Shopify 搭建），不依赖第三方平台 |
| Dropshipping | — | 无货源模式，订单产生后再向供应商下单直接发给消费者，无需囤货 |

## 广告术语

| 术语 | 英文 | 解释 |
|------|------|------|
| ACOS | Advertising Cost of Sales | 广告销售成本比 = 广告花费 / 广告销售额，越低越好 |
| ROAS | Return on Ad Spend | 广告支出回报率 = 广告销售额 / 广告花费，ROAS = 1/ACOS |
| TACoS | Total Advertising Cost of Sales | 总广告销售成本比 = 广告花费 / 总销售额（含自然单） |
| CPC | Cost Per Click | 每次点击费用，广告竞价的基本单位 |
| CTR | Click Through Rate | 点击率 = 广告点击次数 / 广告曝光次数 |
| CVR | Conversion Rate | 转化率 = 成交订单数 / 点击次数 |
| Impression | — | 曝光量，广告被展示的次数 |
| SP 广告 | Sponsored Products | 赞助商品广告，最常见的亚马逊广告类型，出现在搜索结果中 |
| SB 广告 | Sponsored Brands | 品牌推广广告，出现在搜索结果顶部，需品牌注册 |
| SD 广告 | Sponsored Display | 展示型推广广告，可定向竞品页面和站外受众 |
| 自动广告 | Auto Campaign | 亚马逊自动匹配关键词的广告活动，适合新品测词 |
| 手动广告 | Manual Campaign | 卖家手动指定关键词或商品定投的广告活动 |
| 精准匹配 | Exact Match | 广告仅在买家搜索词与关键词完全一致时触发 |
| 广泛匹配 | Broad Match | 广告在搜索词包含关键词相关词汇时均可触发，覆盖面广 |
| 词组匹配 | Phrase Match | 搜索词必须包含关键词短语，介于精准和广泛之间 |
| 否定关键词 | Negative Keywords | 排除不希望触发广告的词，避免无效花费 |
| 竞价 | Bid | 参与广告竞拍时愿意支付的最高 CPC 金额 |
| ROAS 目标 | Target ROAS | 广告系统自动调整出价以达到设定 ROAS 目标 |

## 物流术语

| 术语 | 英文 | 解释 |
|------|------|------|
| 头程 | First-Mile Logistics | 从国内工厂/货源地发货到海外仓（亚马逊仓库）的物流段 |
| 尾程 | Last-Mile Delivery | 从海外仓到买家手中的最后一段配送 |
| 海运 | Sea Freight | 通过集装箱船运输货物，成本低，周期约 25-45 天 |
| 空运 | Air Freight | 通过航班运输货物，速度快，费用较高 |
| 快递 | Express | DHL、UPS、FedEx 等国际快递，适合小批量时效性强的货物 |
| 专线物流 | Dedicated Line | 货代整合的特定航线方案，价格和时效介于空运和海运之间 |
| 三方仓 | Third-Party Warehouse | 非亚马逊的第三方海外仓储服务商 |
| 报关 | Customs Clearance | 向目的国海关申报货物信息并缴纳关税的过程 |
| HS Code | Harmonized System Code | 海关编码，国际通用的商品分类编码，决定关税税率 |
| 完税价格 | DDP（Delivered Duty Paid） | 含税交货，卖家承担目的国所有关税和费用 |
| 清关未税 | DDU（Delivered Duty Unpaid） | 买方承担关税，现已被 DAP 取代（需核实） |
| POD | Proof of Delivery | 签收凭证，证明货物已成功送达买家 |
| ETA | Estimated Time of Arrival | 预计到达时间 |
| LCL | Less than Container Load | 拼箱，货量不足整箱时与他人拼柜共享集装箱 |
| FCL | Full Container Load | 整柜，独立使用一个集装箱 |

## 财务术语

| 术语 | 英文 | 解释 |
|------|------|------|
| GMV | Gross Merchandise Volume | 商品交易总额，包含退款和未结算订单 |
| Net Revenue | — | 净收入 = GMV - 退货退款 |
| 毛利率 | Gross Margin | 毛利润 / 销售额，毛利润 = 销售额 - 商品成本 - 物流成本 - 平台佣金 |
| ROI | Return on Investment | 投资回报率 = 利润 / 投入成本 |
| Referral Fee | — | 亚马逊平台佣金，通常为销售额的 6%-15%（按品类不同） |
| FBA Fee | — | FBA 配送费，按商品重量和尺寸收取 |
| 月仓储费 | Monthly Storage Fee | 亚马逊按库存占用空间和时间收取的仓储费用 |
| 长期仓储费 | Long-term Storage Fee | 库存滞留超过 365 天后额外收取的仓储费用 |
| 退款率 | Refund Rate | 退款订单数 / 总订单数，影响账号健康 |
| ODR | Order Defect Rate | 订单缺陷率，包含差评、A-to-Z 索赔等，需控制在 1% 以下 |
| 结汇 | Foreign Exchange Settlement | 将外币（如美元）兑换成人民币的操作 |
| Payoneer | — | 常用跨境收款工具之一，支持从亚马逊收取销售款 |
| PingPong | — | 国内跨境收款平台（需核实当前运营状态） |
| VAT | Value Added Tax | 增值税，销往欧洲等地区需注册当地 VAT |
| 汇率风险 | Exchange Rate Risk | 外币结汇时因汇率波动导致收入变化的风险 |

> **说明：** 标注「需核实」的数据建议以官方最新文档为准，平台政策可能随时调整。
